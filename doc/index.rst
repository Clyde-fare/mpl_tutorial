Downloads
=========

- :download:`notebook tarball <_static/notebooks.tgz>`

The notebook tarball contains two directories, `notebooks` and `notebooks_ex`.
`notebooks` is the source of the information on this page: all code is filed-in
and plots are shown.  `notebooks_ex` are only partially filled-out, and meant
for interactive use during the live tutorial.
